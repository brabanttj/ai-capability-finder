Add how you AI

Got a task you automate with AI that isn't here yet? Capabilities live in one spreadsheet, and publishing an update takes a couple of clicks.

The 3 steps

  • 1 Open and edit ai_automation_master.xlsx in the project folder.
  • 2 Save it and close Excel.
  • 3 Double-click publish-changes.bat. Wait 1–2 minutes and refresh this site.

What goes in the spreadsheet

The workbook has two tabs:

TabWhat it holdsKey columns
Capabilities One row per task Department, Category, Capability, Description, then an X under each AI tool that applies
Tools One row per AI tool Tool, Icon, Category, Available (Yes/No), DocURL (the “View docs” link)

Add a new capability

  1. On the Capabilities tab, go to the first empty row.
  2. Fill in Department, Category, Capability, and Description.
  3. Put an X under each AI tool that's enabled for it.
  4. Save, close Excel, and run publish-changes.bat.

Add a brand-new AI tool

This needs two edits, because tools live on both tabs:

  1. On the Tools tab, add a row — set Available to Yes and paste the doc link.
  2. On the Capabilities tab, add a column with the exact tool name, then X the rows where it applies.
💡 Tip: Keep the file name, the two tab names, and the main column names exactly as they are, and always close Excel before you publish. A tool only shows for a capability when it has an X and its Available is Yes.
